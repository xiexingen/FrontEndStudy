# 包和模块

## 项目(Package)

很多语言里面也叫 Project，可以用来构建、测试和分享包

## 工作空间(WorkSpace)

对于大型项目，可以进一步将多个包联合在一起，组织成工作空间

## 包(Crate)

一个由多个模块组成的树形结构，可以作为三方库进行分发，也可以生成可执行文件进行运行

## 模块(Module)

很多语言里面叫 Package 可以一个文件多个模块，也可以一个文件一个模块，模块可以被认为是真实项目中的代码组织单元
