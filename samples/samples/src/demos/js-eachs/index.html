js 遍历总结

含原型属性 含不可枚举属性 含Symbol属性 返回值
for...in 是 否 否 key
Object.keys 否 否 否 [key...]
Object.getOwnPropertyNames 否 是 否 [key...]
Object.getOwnPropertySymbols 否 是(只有symbol的) 是 [key...]
Reflect.ownKeys 否 是 是 [key...]
Object.values 否 否 否 [value...]
Object.entries 否 否 否 [[key,value]...]
  • 只有for...in 可以遍历到原型上的属性
  • Object.getOwnPropertyNames和Reflect.ownKeys可以获取到不可枚举的属性
  • Ojbect.getOwnPropertySymbols和Reflect.ownKeys可以获取到Symbol属性
