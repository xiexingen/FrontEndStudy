# 所有权&借用

在其他语言，如 C# 中，内存安全几乎都是通过 GC 的方式实现，但是 GC 会引来性能、内存占用以及 Stop the world 等问题。而 Rust 采用了`所有权系统`的方式。

## 栈

先进后出、已知且固定大小的内存空间

## 堆

大小未知或者可能变化的数据存储在堆上，并返回一个表示该位置地址的`指针`存储到`栈`上
